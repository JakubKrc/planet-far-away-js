Planet Far Away Editor prealpha version -0.00010065484


disclaimer: nobody should use this

WSAD: moving screen
Left Click: set selected tile. Beware!: in tiles there are also not just tiles.
Q: toggle collisionbox rendering.
You can erase tile by selecting 11. tile.
By clicking write it will generate js code in green box hidden down, you need to scroll there.
You can try load levels (level1, level2)
Use scroll wheel to scroll tiles. Have to hover above tile selection.
Right click will put into level object you have selected in left middle table.
You can drag object by clicking and holding right mouse button.
You can change image to any object by changing imagenum in bottom left table. Have to click 'saveV'.
'dimensions' button is for setting dimensions to selected object according to its image. Then you have to 'saveV'.
And so on. It is possible to use this but its quite glitchy and unintuitive. But was used to create all playable levels.

Episode select is currently hardcoded in html.
